# Smart Building Compliance Checker

An AI-powered architectural compliance agent that automatically evaluates BIM (.ifc) models against building codes and regulatory PDFs using Agentic Workflow and 100% accurate Pandas code execution.

**Key Features:**
* Agentic Tool-Calling AI for zero-hallucination math evaluations
* Smart RAG ingestion of PDF Building Codes (German NBauO, IBC, etc.)
* Dynamic IFC Property Flattening (FireRatings, Materials, Acoustics)
* Beautiful Markdown validation reports locating precise `GlobalId`s

*Developed for next-generation BIM verification.*
